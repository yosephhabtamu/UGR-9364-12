  1. why is software development hard?

    answer

    In my opinion, a lot of people who say "software development is hard" can be catagorized into three group. The first group are the one who does not know the difference between developer and coder. These people think software development as a hard task because they jump right into coding without planning ahead, stating their software requirement and without proper design. They are like a ship without a compass. My advice for these people is a program can do what ever you want the sky is the limit so you have to know your exact destination before starting your journey.

  2. what is the best programming language for beginners?

    answer

    I think there is no perfect programming language for every one. what I recommend is trying to understand the common features all programming languages share. For example: variables, data types, loops, if statements , classes and objects. uderstanding all these concepts very well is far more better than knowing 3 or 4 languages. so if these concepts are mastered then it is a matter of memorizing syntax like a nursery rhyme.

  3. what is your advice for people who want to start learning programming?

    answer

    first, select one language and master the concepts like variables, classes and objects not the syntax. Then start your own project and always apply what you learned on the project.
